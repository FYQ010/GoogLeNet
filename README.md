# GoogLeNet
- 基于GoogLeNet进行年龄和性别分类

- 本项目采用人脸数据集UTKFace Large Scale Face Dataset，下载链接为：https://susanqq.github.io/UTKFace/.

- 人脸识别使用Dlib

- 年龄标签为：
    "0": "0_10",
    "1": "11_20",
    "2": "21_30",
    "3": "31_40",
    "4": "41_50",
    "5": "51_60",
    "6": "61_70",
    "7": "71_80",
    "8": "81_90",
    "9": "over_90"

- 性别标签为：
    "0": "female",
    "1": "male"
